# Goal

Create an app that displays a list of movies with the ability to compare the similarities of two movies.

## Tasks

Try to work on one task after the other:

1. First use the `getPage()` function in the [src/api/content.ts](./src/api/content.ts) to list all movies in the form of a grid. _(Use the image [Wireframe1.png](./reference/Wireframe1.png) as reference)_
2. Create a section where two movies are compared and you can see if they are similar or not. _(Use the image [Wireframe2.png](./reference/Wireframe2.png) as reference)_

## Pointers

- Ask questions if anything is unclear.
- Check the wireframe images for reference ([src/reference/](./reference/)).
- Aim for responsive design.
- Work with the `getPage()` function inside `content.ts` for data fetching, avoid working directly with the JSON file.
- "products" in the `data.json` are the movies objects in the wireframe images.
